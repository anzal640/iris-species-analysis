What's inside:

The Analysis: I spent time cleaning the data and making some charts with Matplotlib and Seaborn to see how things like age and blood pressure play a role.

The Tools: I did all the work in Google Colab using Python, Pandas, and I've imported Sklearn to get ready for some machine learning later on.
